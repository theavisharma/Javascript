Function Declaration -

If the user have to create a function then , they have to must declare it and write its functionalities.

Function Calling -

User have to call the function to use it.

Local Variable -

Variable which is declared inside the function is known as local Variable. It is only used inside function.

Outer Variable -

Variable which is declared outside the function is known as outer Variable. It is also used outside the function.

Parameters -

Function parameters are listed inside the parentheses () in the function definition.
Function arguments are the values received by the function when it is invoked.
Inside the function, the arguments (the parameters) behave as local variables.
