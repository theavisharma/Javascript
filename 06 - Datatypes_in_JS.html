A value in JavaScript is always of a certain type. For example, a string or a number.
There are eight types of datatypes in JavaScript

1. NULL - Null is used to represent empty value or null value.

2. Number - Number is used to represent both number and floating value.

3. Symbol - A symbol is used to declare a symbol.

4. String - A string is used to store a character set.

5. Boolean - Booleans are often used in conditional testing.

6. BigInt - BigInt is used to store a large integral value.

7. Undefined - A variable without a value, has the value undefined.

8. Object -

Typeof Operator - Typeof operators is used to return type of datatypes(return a string)
